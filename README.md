


# Projeto Calculadora / Calculator Project

Este projeto representa o estágio inicial de aprendizado em Python, focado em lógica de programação, estruturas condicionais e controle de versão com Git e GitHub.

*This project represents the initial stage of Python learning, focusing on programming logic, conditional structures, and version control with Git and GitHub.*

---

## Instruções de Execução / How to Run

Para executar o projeto, é necessário ter o Python 3 instalado. Utilize os comandos abaixo no terminal:

*To run the project, Python 3 must be installed. Use the following commands in the terminal:*

* `python calculator.py´

---

## Conceitos Aplicados / Concepts Applied

* **Lógica e Condicionais:** Implementação de operações aritméticas através de estruturas `if`, `elif` e `else`.
  *(Logic & Conditionals: Implementation of arithmetic operations through if, elif, and else structures.)*
* **Manipulação de Dados:** Captura e conversão de tipos de dados de entrada do usuário.
  *(Data Input: Capturing and converting user input data types.)*
* **Controle de Versão:** Gestão de repositório e histórico de commits via Git.
  *(Version Control: Repository management and commit history via Git.)*
